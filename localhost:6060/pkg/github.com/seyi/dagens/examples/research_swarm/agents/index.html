func BuildDemoModelProviderFromEnv ¶ 

func BuildDemoModelProviderFromEnv() (model.ModelProvider, func() error, bool, error)

BuildDemoModelProviderFromEnv builds a model.ModelProvider used by internal LlmAgents. It prefers real LLM credentials; if absent, it falls back to a deterministic mock provider unless RESEARCH_SWARM_REQUIRE_LLM=true.
